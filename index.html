二核心概念速查

1. 专门用途中文（线索一）

专门用途中文（CSP）以学习者特定需求为起点，面向学术、职业或行业任务，重新组织课程目标、内容、方法与评价的教学取向；性质上是一种“通道课程”。
“中文+” / “+中文”中文教育与专业、技能、职业教育融合的宏观战略与项目模式；“+中文”则以专业学习居于主位、中文提供语言支持。
目标语言 / 工具语言中文本身是学习目的时为目标语言（通用中文、专门用途中文皆属此）；中文是学习其他内容的手段时为工具语言（“中文+X”）。
学术用途中文 / 行业用途中文按学习者所处领域与任务划分：前者服务专业学习与学术交流，后者服务具体岗位与业务场景。
逆向设计 / 表现性测评先定真实任务目标与评价证据，再设计教学；以真实或仿真任务中的表现来评价学习成效。

2. 区域国别中文教育（线索二）

区域国别学以特定国家或区域为对象、围绕现实问题综合多学科与在地材料的交叉学科；“区域”重跨国关联，“国别”重一国整体。
区域国别中文教育对中国以外特定区域或国家中文教育的系统研究，关注其在当地教育体系、语言生态与政策环境中的运行，即对象国本地的中文教育生态。
全球中文教育同心圈按中文作为何种通用语来教学，分内圈（语文教育）、中圈（华文教育）、外圈（中文作外语）；国际中文教育＝中圈＋外圈。
两种基本形态华文教育（中文多具母语、祖语属性）与中文作为外语·第二语言教育；二者可在同一国家、学校或课堂中并存。
本土化的四个层级表层（文化素材）→中层（主题任务与学习者画像）→深层（目标、评价与师资对接）→制度层（进入国民教育与修订机制）。

三关键框架速记

  • 三种课程模式：任务型／内容依托式（含 CBI、CLAC）／速成型岗位中文——按课程主体、学习者水平与资源条件选择。
  • 大纲两种取向：语言要素型（《国际中文教育中文水平等级标准》、HSK 3.0）↔ 能力描述型（ACTFL、CEFR 的“能做”）；多数大纲在二者之间。
  • 标准对接四个方面（王祖嫘 2024）：教学目标 · 教学内容 · 教学评价 · 语言认知。
  • 国别大纲研制十步流程：生态分析 → 学习者分类 → 课程定位 → 标准对接 → 目标分级 → 主题与语言要素组织 → 资源配套 → 评价设计 → 试点反馈 → 修订制度化（→ 回到生态分析，周期循环）。

四数字工具与电子资源

  1. 语料检索与分析工具：BCC、CCL 语料库，AntConc、Sketch Engine、Wmatrix7/PyMUSAS。可做频率、关键词、N 元语块、搭配与语义域分析；中文自动标注仍需人工复核，课堂中宜作为“发现线索”的工具。
  2. 商务与行业中文资源：国际中文教育资源网的商务汉语词表、功能项目语料与教材文本库；对外经济贸易大学“商务中文资源”及“商务中文 Pal”小程序，可用于编制任务词表、功能表达与教材例句。
  3. 自建小语料库工具链：语料采集 → 脱敏与元数据记录 → 分词与自定义词典 → 频率·关键词·搭配·类联接·语步分析 → 行业专家与教师复核 → 转化为词表、句式表、任务链与评价量规。
  4. 虚拟仿真与多模态资源：航空、医学、工程等高风险或强操作领域，可将 VR/AR、虚拟标准化病人、虚拟客舱、虚拟工厂等做成情境任务资源包；资源须服务任务目标、语言产出与评价，而非只展示场景。
  5. 学习者数据与评价资源：平台日志、口语录音、写作修改稿、角色扮演视频、同伴与企业导师评价，可形成学习者语料库；其价值在于反向修订教材——发现哪些词句未掌握、哪些任务过难、哪些评价指标需调整。
  6. 标准与大纲参照：《国际中文教育中文水平等级标准》（GF0025—2021）、《职业中文能力等级标准》（T/ISCLT 002—2023）、HSK 3.0；以及 ACTFL、CEFR 等海外能力框架，供大纲研制与标准对接时检索比对。
